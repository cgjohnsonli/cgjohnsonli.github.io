代码优化：
```python
def factorial(n):
    """计算n的阶乘
    
    Args:
        n: 非负整数
    Returns:
        n的阶乘结果
    """
    # 参数验证
    if not isinstance(n, int) or n < 0:
        raise ValueError("输入必须是非负整数")
    
    # 计算阶乘
    result = 1
    for i in range(1, n + 1):  # 直接从1开始
        result *= i
    return result

try:
    num = int(input("请输入一个数："))
    print(f"{num}的阶乘是：{factorial(num)}")
except ValueError as e:
    print("输入错误：", e)
```

改进点说明：
1. 添加函数文档说明
2. 增加参数类型检查
3. 优化循环范围设置
4. 添加异常处理机制
5. 简化乘法运算符使用
