案例二：练习题生成

场景描述

为Python课程生成不同难度的练习题，包括选择题、填空题和编程题。

人类输入

要求：
1. 不同题型设计
2. 难度层次划分
3. 知识点覆盖
4. 提供参考答案
AI回复
为您生成的练习题如下：

1. 选择题（基础）

Q1: 以下哪个不是Python的循环语句？
A. for
B. while
C. loop
D. while...else
答案：C
解析：Python主要的循环语句是for和while

Q2: 关于break语句，以下说法正确的是：
A. 只能用于for循环
B. 可以用于for和while循环
C. 只能用于while循环
D. 不能用于循环结构
答案：B
解析：break可用于for和while循环，用于提前结束循环

2. 填空题（中等）

Q1: 使用_____函数可以生成一个整数序列。
答案：range
解析：range(start, stop, step)可生成数字序列

Q2: while循环中，如果条件始终为True，会形成_____。
答案：死循环/无限循环
解析：条件永真会导致循环无法终止

4. 编程题（进阶）

Q1: 编写程序判断一个数是否为质数
```python
# 参考答案
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True

num = int(input("输入一个数："))
if is_prime(num):
    print("是质数")
else:
    print("不是质数")
```

Q2: 编写程序实现斐波那契数列的前20项
```python
# 参考答案
def fibonacci(n):
    a, b = 0, 1
    for i in range(n):
        print(a, end=" ")
        a, b = b, a + b

fibonacci(20)
```

5. 综合应用题（挑战）

Q1: 编写一个简单的ATM程序
要求：
- 初始余额1000元
- 支持查询、存款、取款功能
- 取款金额不能超过余额
- 输入q退出程序

```python
# 参考答案
balance = 1000

while True:
    print("\n1-查询 2-存款 3-取款 q-退出")
    choice = input("请选择：")
    
    if choice == 'q':
        break
    elif choice == '1':
        print(f"余额：{balance}元")
    elif choice == '2':
        amount = float(input("存款金额："))
        if amount > 0:
            balance += amount
            print("存款成功")
    elif choice == '3':
        amount = float(input("取款金额："))
        if amount > balance:
            print("余额不足")
        elif amount > 0:
            balance -= amount
            print("取款成功")
```
