命令和参数

《创意设计》AI辅助设计

引言

本页面详细介绍了Midjourney提供的各种命令和参数，帮助您更好地控制和优化图像生成过程。内容结构基于Midjourney官方文档，为您提供全面而准确的信息。

1. 命令

1.1 命令列表

以下是Midjourney机器人支持的命令列表。这些命令可以在任何机器人频道、允许Midjourney机器人操作的私人Discord服务器上，或与Midjourney机器人的直接消息中使用。

  • /ask：获取问题的答案。
  • /blend：轻松混合两张图片。
  • /daily_theme：切换#daily-theme频道更新的通知提醒。
  • /describe：基于您上传的图片生成四个示例提示。
  • /fast：切换到快速模式。
  • /help：显示有关Midjourney机器人的基本信息和提示。
  • /imagine：使用提示生成图像。
  • /info：查看您的账户信息以及任何排队或运行中的作业。
  • /prefer option set：创建或管理自定义选项。
  • /prefer option list：查看您当前的自定义选项。
  • /prefer remix：切换重组模式。
  • /prefer suffix：指定要添加到每个提示末尾的后缀。
  • /public：（仅限Pro计划订阅者）切换到公共模式。
  • /relax：切换到放松模式。
  • /settings：查看和调整Midjourney机器人的设置。
  • /stealth：（仅限Pro计划订阅者）切换到隐身模式。
  • /shorten：提交长提示并获得如何使其更简洁的建议。
  • /show：使用图像的作业ID在Discord中重新生成作业。
  • /subscribe：生成用户账户页面的个人链接。
  • /tune：基于您的提示生成风格调谐器。风格调谐器允许您创建自己的Midjourney风格并自定义作业的外观。
  • /turbo：切换到涡轮模式。
  • /userid：获取您的Midjourney用户ID。

注意：一些命令（如/docs、/feedback、/faq、/invite、/list_tuners、/synonyms）仅在官方Midjourney Discord服务器中可用。

已弃用的命令

  • /private（已被'/stealth'替代）
  • /pixels
  • /idea

1.2 blend

/blend命令允许您快速上传2-5张图片，然后分析每张图片的概念和美学特征，将它们融合成一张全新的图像。

  • /blend的效果等同于在/imagine命令中使用多个图像提示，但其界面针对移动设备进行了优化，使用更加便捷。
  • /blend最多支持5张图片。如果需要使用超过5张图片，请使用图像提示配合/imagine命令。
  • /blend不支持文本提示。如果需要同时使用文本和图像提示，请使用图像提示和文本配合/imagine命令。

/blend命令选项

输入/blend命令后，系统会提示您上传两张照片。您可以从硬盘拖放图片，或在移动设备上从照片库添加图片。如需添加更多图片，请选择optional/options字段，然后选择image3、image4或image5。请注意，/blend命令可能比其他命令启动时间更长，因为系统需要先上传您的图片，然后Midjourney机器人才能处理您的请求。

融合后的图像默认使用1:1的宽高比，但您可以使用可选的dimensions字段选择正方形（1:1）、肖像（2:3）或风景（3:2）宽高比。

自定义后缀会被添加到/blend提示的末尾，就像其他/imagine提示一样。在/blend命令中指定的宽高比会覆盖自定义后缀中的宽高比设置。

融合技巧

为获得最佳效果，请上传与您期望的结果具有相同宽高比的图像。

如何使用/blend

1.3 describe

/describe命令允许您上传一张图片，并基于该图片生成四个可能的提示。使用/describe命令来探索新的词汇和美学风格。

  • describe功能生成的提示建议具有启发性和暗示性。它不能用于精确重现上传的图像。
  • describe在Discord和网站上的工作方式略有不同。在Discord中使用/describe命令会生成完整的提示，而网站则会提供词语和短语建议，帮助您构建自己的提示。
  • 对同一图像多次使用"describe"会提供不同的建议。

在Discord中使用

使用/describe命令

在Discord中任何可以使用Midjourney机器人的地方使用/describe命令。您可以通过输入图片的URL（选择link选项；在这里查找更多关于在Discord中使用图像URL的信息），或选择存储在您设备上的图片（使用image选项）来提供要描述的图片。

生成建议后，点击相应的数字按钮来使用该提示/imagine。您可以在提交之前编辑文本。

您还可以使用🔄按钮对同一图像再次运行/describe命令，或使用🎉 Imagine all按钮一次性生成所有四个提示的图像（但您将没有机会先编辑它们）。

Discord中的/describe示例

如果您使用/describe命令描述下面的图像，Midjourney机器人将建议四个可能生成类似图像的提示：

1️⃣ 在白色背景上用紫色圆珠笔绘制的波斯猫脸部素描，紫色毛发，未知艺术家风格。

2️⃣ 紫色铅笔绘制的波斯猫，白色背景，John Beastman风格。

3️⃣ 紫色铅笔绘制的可爱波斯猫，白色背景上的简单线条，具有精细细节和精美构图，超写实风格，锐利的边缘和清晰的焦点。

4️⃣ 紫色铅笔绘制的异国短毛猫，简单的线条艺术和白色背景，精细线条和柔和色彩营造出柔和细腻的氛围，这幅特写肖像。素描风格将猫描绘得可爱而梦幻，通过详细的线条展现复杂细节，正面视角。水墨技法使用低饱和度的深黑色调创造艺术家风格。

旧版/describe
  • 默认情况下，Discord中的/describe生成最适合版本6的提示。
  • 如果您在Discord中将默认Midjourney版本设置为5.0，/describe将生成更短的提示，这些提示更适合较旧的模型版本。

在网站上使用

从想象栏使用

点击想象栏中的图标来访问网站上的图像参考选项。如果您想描述的图像没有出现，可以从此视图上传。上传后，将鼠标悬停在图像上以显示图标，然后点击它来生成建议。

与Discord中的/describe生成完整提示不同，网站会提供各种词语和短语供您使用。点击生成的短语会将其添加到您当前提示的末尾。点击重新生成按钮来生成新的描述，点击返回按钮返回图像选择工具。

从工具标签页使用

点击侧边栏底部的工具菜单来访问Describe页面。该页面将自动填充您所有的已上传图像。

您可以将图像从计算机拖放到页面上，以生成完整提示的建议，类似于在Discord中使用/describe的效果。

将鼠标悬停在提示上，点击+copy按钮将其复制到剪贴板。

1.4 用户信息（User Info）

使用/info命令可以查看关于您当前排队和运行中的作业、订阅类型、续订日期等信息。

用户ID（User ID）

您的Midjourney用户ID

订阅（Subscription）

订阅部分显示您订阅的计划以及下一次续订日期。

可见性模式（Visibility Mode）

显示您当前是处于公开模式还是隐身模式。隐身模式仅适用于Pro计划订阅者。

剩余快速时间（Fast Time Remaining）

显示您本月剩余的快速GPU时间。快速GPU时间每月重置，不会结转。

评分计数（Ranking Count）

显示您评分图像的总数，以及最近30天内评分的图像数量。

终身使用情况（Lifetime Usage）

显示您的Midjourney终身统计数据。图像包括所有类型的生成（初始图像网格、放大、变体、重组等）。

快速、涡轮和放松模式使用情况（Fast, Turbo and Relaxed Usage）

显示您快速、涡轮和放松模式的使用情况。

排队的快速和放松作业（Queued Fast and Relaxed Jobs）

列出所有排队等待运行的作业。

运行中的作业（Running Jobs）

列出所有当前运行的作业。同时最多可以运行三个作业。

使用Info命令

在任何机器人频道或您的私信中输入/info。只有您能看到您的信息弹窗。

作业（Job）是指使用Midjourney机器人的任何操作。作业包括使用/imagine创建初始图像网格、放大图像和创建图像变体。

机器人频道（Bot Channels）是允许Midjourney机器人处理斜杠命令的Discord频道。在官方Midjourney Discord服务器上，新手房间、主题频道、主题图像生成和通用图像生成类别中的所有频道都是机器人频道。

1.5 设置和预设（Settings and Presets in Discord）

/settings命令提供了常用选项的切换按钮，如模型版本、风格值、质量值和放大器版本。设置中还包括/stealth和/public命令的切换。

在提示末尾添加的参数将覆盖使用/settings进行的选择。

模型版本下拉菜单

选择生成图像时使用的Midjourney模型版本。使用最新模型是默认选项，它将始终使用最新的Midjourney模型版本。

风格原始参数

Midjourney模型版本5.1、5.2、6和Niji 6可以使用--style raw参数进行微调，以减少Midjourney默认美学。如果选择了其他模型版本，则此切换不可用。

🔧 Raw Mode

风格化参数

Midjourney机器人已经过训练，可以生成偏好艺术色彩、构图和形式的图像。--stylize或--s参数影响这种训练的应用强度。低风格化值生成的图像与提示更加匹配，但艺术性较低。高风格化值创建的图像非常具有艺术性，但与提示的联系较少。

🖌️ Stylize Low 🖌️ Stylize Med 🖌️ Stylize High 🖌️ Stylize Very High

Stylize Low = --s 50, Stylize Med = --s 100, Stylize High = --s 250, Stylize Very High = --s 750

公开和隐身模式

在公开和隐身模式之间切换。对应/public和/stealth命令。

重组模式

使用重组模式在变体之间更改提示、参数、模型版本或宽高比。重组将利用您起始图像的一般构图，并将其作为新作业的一部分。重组可以帮助改变图像的场景或光线、进化主题或实现复杂的构图。

🎛️ Remix

高变化和低变化模式

在高变化和低变化模式之间切换。

粘性风格

粘性风格会将上次使用的--style代码参数保存在您的个人后缀中，这样您就不必在未来的提示中重复该代码。通过使用新的--style或取消选择粘性风格来更改代码。

📌 Sticky Style

涡轮、快速和放松模式

在涡轮、快速和放松模式之间切换。对应/turbo、/fast和/relax命令以及--turbo、--fast和--relax参数。

⚡ Turbo Mode 🐇 Fast Mode 🐢 Relax Mode

Reset Settings

自动私信结果

使用/prefer auto_dm来开启或关闭自动将完成的作业发送到您的私信。

自定义偏好

使用prefer命令创建自定义选项，以自动将常用参数添加到提示的末尾。

  • /prefer suffix 指定要添加到每个提示末尾的后缀。
  • /prefer option 创建或管理自定义选项。
  • /prefer option list 查看您当前的自定义选项。

偏好后缀

/prefer suffix自动在所有提示后附加指定的后缀。

命令示例：/prefer suffix --ar 16:9 --video

要清除设置的后缀，使用/settings命令并选择 Reset Settings，或再次使用/prefer suffix而不添加可选的new_value字段到命令中。

选项集

/prefer option set <name> <value>命令允许您创建自定义快捷方式，将多个参数或文本片段添加到您的提示中。

/prefer option set mine --chaos 5 --aspect 2:3 创建了一个名为"mine"的选项，其中包含chaos和宽高比参数。

使用/imagine prompt vibrant California poppies --mine，会被解释为/imagine prompt vibrant California poppies --chaos 5 --aspect 2:3。

  • 您可以在同一个提示中使用多个选项集。
  • 您可以在选项集中混合使用提示文本和参数。如果这样做，集合中的所有参数必须放在末尾，就像正常提示一样。
  • 您的选项集将在提示中放置的位置"展开"，所以您需要将所有提示参数放在选项集之后。

/prefer option list 列出所有使用prefer option set创建的选项。用户最多可以拥有20个自定义选项。

要删除自定义选项，使用/prefer option set <要删除的名称>而不添加可选的value字段到命令中。

订阅者可以在Discord的私信中一对一地与Midjourney机器人工作，而不是在公共频道中。在您的私信中创建的图像仍然受内容和审核规则的约束，并将在您的Midjourney网站画廊中可见。

1.6 显示作业（Show Job）

您可以使用/show命令和唯一的作业ID来将作业移动到另一个服务器或频道、恢复丢失的作业，或刷新旧作业以创建新的变体、放大，或使用较新的参数和功能。

/show只适用于您自己的作业。

查找作业ID

作业ID是Midjourney生成的每个图像的唯一标识符。
作业ID看起来像这样：9333dcd0-681e-4840-a29c-801e502ae424，可以在所有图像文件名的第一部分、网站上的URL中以及图像的文件名中找到。

在网页上

您可以通过选择Options > Copy > Job ID来找到会员画廊中任何图像的作业ID。

从URL中

在Midjourney画廊中查看图像时，作业ID在URL中。
https://www.midjourney.com/jobs/9333dcd0-681e-4840-a29c-801e502ae424?index=0

从文件名中

查看从画廊下载的图像时，作业ID是文件名的最后一部分。
User_cat_cloud_spirit_9333dcd0-681e-4840-a29c-801e502ae424.png

使用Discord表情反应

对您完成的作业用信封表情✉️进行反应，Midjourney机器人会向您发送一条包含其作业ID的私信。如果您没有收到机器人的私信，请检查您的Discord隐私设置。

如何使用Show命令

在任何机器人频道中使用/show <作业ID #>来恢复作业。

机器人频道是允许Midjourney机器人处理斜杠命令的Discord频道。在官方Midjourney Discord服务器上，新手房间、主题频道、主题图像生成和通用图像生成类别中的所有频道都是机器人频道。

1.7 缩短（Shorten）

/shorten命令分析您的提示，突出显示一些最有影响力的词汇，并建议可以删除的不必要词语。通过这个命令，您可以通过专注于核心术语来优化您的提示。

/shorten与多重提示或--no参数不兼容

使用/shorten分析提示

Midjourney机器人通过将提示分解成更小的单位（称为标记）来分析它。这些标记可以是短语、单词甚至音节。Midjourney机器人将这些标记转换为它能理解的格式。它将它们与在训练过程中学到的关联和模式一起使用，来指导图像的生成方式。可以将标记视为帮助Midjourney机器人理解输入并创建所需视觉输出的构建块。

包含不必要词语、冗长描述、诗意短语或直接对机器人说话（"请为我制作一张图像"，"谢谢你的帮助，Midjourney机器人！"）的长提示可能会导致意外元素被添加到您的图像中。

/shorten命令可以帮助您发现提示中最重要的词语，以及可以省略的词语。

/shorten命令示例

如果您想创建一堆撒满糖霜的甜甜圈，您可能会尝试使用以下提示：
请创建一个奇幻壮丽的甜甜圈塔，精心制作并装饰有令人着迷的多彩糖霜。将这个糖果杰作栩栩如生地呈现出来，确保每个细节都以令人惊叹的魔幻现实主义风格呈现。谢谢！

如果您对上述提示使用/shorten命令，Midjourney机器人将返回以下信息：

重要标记
请创建一个奇幻壮丽的塔的甜甜圈，精心制作并装饰有令人着迷的多彩糖霜。将这个糖果杰作栩栩如生地呈现出来，确保每个细节都以令人惊叹的魔幻现实主义风格呈现。谢谢！

缩短的提示
1️⃣ 请，壮丽的甜甜圈塔，制作，多彩糖霜数组，糖果杰作，呈现，魔幻现实主义 2️⃣ 请，壮丽的甜甜圈塔，多彩糖霜，糖果杰作，呈现，魔幻现实主义 3️⃣ 壮丽的甜甜圈塔，多彩糖霜，糖果，魔幻现实主义 4️⃣ 壮丽的甜甜圈塔，多彩糖霜，魔幻 5️⃣ 甜甜圈塔，糖霜

提示中最重要的标记以粗体显示，最不重要的以删除线显示。您还将得到基于这些信息的5个可能的较短提示。

分析结果

最短的提示，选项5️⃣：甜甜圈塔，糖霜生成的图像最接近原始目标。许多填充词如"奇幻"、"令人着迷的"和"杰作"可以省略。了解到"塔"和"魔幻"被认为是重要标记有助于解释为什么一些图像生成时带有童话城堡元素。了解这一点提供了一个线索，如果目标是创建一堆美味的甜甜圈，应该从提示中删除"魔幻"。

/shorten命令是一个工具，可以帮助您探索Midjourney机器人如何解释标记并实验词语，但它可能不适用于所有主题和提示风格。

如何使用Shorten

在任何机器人频道中使用/shorten <您的提示>命令来获取关于您提示的信息

机器人频道是允许Midjourney机器人处理斜杠命令的Discord频道。在官方Midjourney Discord服务器上，新手房间、主题频道、主题图像生成和通用图像生成类别中的所有频道都是机器人频道。
